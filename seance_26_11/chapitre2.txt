Chapitre 2 : Les alternatives, les structures de contrôle

2.1. L'alternative simple
syntaxe en algo :
    Si condition
        Alors instruction
    Fin si

Syntaxe dans les langages (C et PHP) :
    if (condition) {
        instruction ;
    }
L'alternative simple exécute une ou plusieurs instructions si une condition est vérifiée. Dans le cas contraire, le bloc d'instruction est simplement sauté.

La condition peut être simple ou composée à l'aide des opérateurs et, ou, non.

Exemple :

    Si note < 0 ou note > 20
        Alors afficher ("la note saisie est erronée.")
    Fin si

    if (note < 0 || note > 20 ){
        printf("la note saisie est erronée.");
    }

2.2. L'alternative double
Syntaxe en algo :
    Si condition
       alors instruction_1
       sinon instruction_2
    Fin si
Syntaxe dans les langages (C et PHP) :
    if (condition){
        instruction_1;
    } else {
        instruction2;
    }
L'alternative double exécute la première instruction ou la deuxième instruction selon que la condition est vérifiée ou non. Il y a une exclusion mutuelle des deux instructions : c'est soit l'une soit l'instruction qui est exécutée.
Mais jamais les deux a la fois.

Exemple :
    Si moyenne >= 10
       Alors afficher("etudiant admis")
       Sinon afficher("etudiant recalé")
    Fin si

    if (moyenne >= 10){
        printf("etudiant admis");
    } else {
        printf("etudiant recalé");
    }

2.3. Le choix multiple
Syntaxe en Algo :
    Selon l'expression faire
      cas valeur_1 : instruction_1
    …
      Sinon instruction_defaut
    FIn selon
Syntaxe dans les langages (C et PHP) :
    switch(expression{
      case valeur_1 : instruction_1; break;
    …
    default : instruction_defaut; break;
    }

Le selon permet une bonne structuration de l'imbrication des alternatives. L'expression doit être de type enuméré (entier ou caractère).

Exemple :
    selon jour faire
      cas 1 : cas 2 : cas 3 : cas 4 : cas 5 :
         afficher("C'est un jour de repos")
          sinon afficher("erreur de saisie")
    Fin selon
swith (jour){
    case 1 : case 2 : case 3 : case 4 : case 5 :
       printf("c'est un jour de travail") ; break;
    case 6 : case 7 :
       printf("C'est un jour de repos"); break;

    default : printf ("erreur de saisie"); break;
}
